# Ramen BBS
## Show your ramen you ate today
今日食べたラーメンを投稿できる掲示板風サイト
* 今日食べたラーメンの店名、URL、都道府県、評価、画像、コメントを投稿できる
* ユーザー登録（メールアドレス、ユーザー名、パスワード）・ログイン機能
* 詳細ページで自分の投稿を編集・削除できる
* 都道府県で投稿の絞り込みができる
* 各ユーザーのプロフィールに累計杯数・直近一週間に食べた日の表示
* FuelPHP, Knockout.js, React
